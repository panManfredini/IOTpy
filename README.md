# IOTpy
Framework to expose IOT devices trough REST API and prometheus metrics

# Installation
```bash
git clone https://github.com/panManfredini/IOTpy
cd IOTpy
pip install .
```

# Run the Example
```bash
cd example
export IOTPY_DEVICES_DIR=$(pwd)
iotpy
```
Now visit [http://localhost:8085](http://localhost:8085).
